Create a new file named __WEBSITE_NAME__.js based on the module_template.js file. Replace the following placeholders:

1. __MODULE_NAME__ with the service name
2. __MODULE_VERSION__ with the current version (format: YY.MM.DD.1)
3. __CLASS_NAME__ with the appropriate class name
4. __JOB_NAME__ with the job function name
5. __HOST_URL__ with the base URL of the service endpoint

Make sure all placeholders are replaced with appropriate values to create a functional module file.

| Parameter | Value |
|-----------|-------|
| __VERSION__ | 0.0.1 |
| __WEBSITE_NAME__ | MIS |
| __CLASS_NAME__ | NOTICE |
| __HOST_URL__ | https://mis.chungbuk.ac.kr |
| __JOB_NAME__ | NOTICE_LIST |
